## Who may use this grant

A prospective employer or client evaluating Wasiliy Strecker for employment or
a contract engagement may use this grant. Use is limited to people directly
involved in that genuine recruitment or technical evaluation.

## Permission

For that evaluation purpose only, the evaluator may:

- download or clone the repository
- reproduce, compile, and run the software locally or in a private internal
  build, CI, or test environment
- inspect the architecture, source code, behavior, and security of the software
- make temporary changes that are reasonably necessary to build or test the
  software
- share evaluation results within the directly involved recruitment and
  technical review team

There is no fixed calendar expiry. Every exercise of these permissions must
remain solely for a genuine recruitment evaluation. Active use must stop when
that purpose ends.

This grant does not permit:

- production use or use for internal business operations

- integration into any product or service
- sale, sublicensing, commercialization, or use for the benefit of a third party
- reuse of temporary evaluation changes in a commercial project
- sharing the software outside the directly involved evaluation team
- security testing against the public FlowForm Studio demo without prior written

Working copies and temporary builds must be deleted when the evaluation ends. A
single inaccessible archival copy may be kept only when retention is required
by law or a binding compliance rule, and it may not be executed or reused.
